Thomas Fence has been the go-to fence contractor in Northwest Arkansas since 1993. We install fences for homeowners, businesses, farms, and contractors throughout Springdale, Bentonville, Rogers, Fayetteville, and the surrounding communities.

Every project is backed by our Arkansas Contractor's License (No. 015501), AFA certification, and a written one-year craftsmanship warranty. We pull utility locates on every job and use concrete for every post — no shortcuts.
